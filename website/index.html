Get Started

# Clone the repository
git clone https://github.com/LyferGang/Botwave.git
cd Botwave

# Install dependencies
pip install -r requirements.txt

# Configure secrets
cp vault/secrets.json.example vault/secrets.json

# Start the dashboard
./start.sh

# Access at http://localhost:5000
