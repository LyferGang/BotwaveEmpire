Get Started

# Clone the repository
git clone https://github.com/LyferGang/BotwaveEmpire.git
cd BotwaveEmpire

# Configure environment
cp .env.example .env
# Edit .env with your settings

# Start with Docker
docker-compose up -d

# Verify deployment
curl http://localhost:8080/health
