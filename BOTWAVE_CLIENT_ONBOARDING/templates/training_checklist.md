# Botwave Client Training Checklist

**Use this during your on-site training session**

---

## Pre-Training Setup

- [ ] Computer cleanup complete
- [ ] Folders created and organized
- [ ] AI agents deployed
- [ ] All software installed
- [ ] Printed materials ready

---

## Training Session (60-90 minutes)

### Part 1: Computer Organization (15 min)

- [ ] Show new desktop layout
- [ ] Explain folder structure
- [ ] Demonstrate file saving workflow
- [ ] Show how to find files
- [ ] Review quick access shortcuts

**Client Practice:**
- [ ] Client saves a file to correct folder
- [ ] Client opens business folders shortcut
- [ ] Client finds a specific file

---

### Part 2: Daily Workflow (20 min)

**Morning Routine:**
- [ ] Turn on computer
- [ ] Open business apps (if configured)
- [ ] Check active jobs folder
- [ ] Review today's schedule

**Evening Routine:**
- [ ] Back up important files
- [ ] Move completed jobs to archive
- [ ] Quick desktop cleanup
- [ ] Proper shutdown

**Client Practice:**
- [ ] Client demonstrates morning routine
- [ ] Client demonstrates evening routine

---

### Part 3: AI Agent Training (20 min)

**Telegram Bot (if deployed):**
- [ ] Show bot link/QR code
- [ ] Demonstrate customer inquiry flow
- [ ] Show quote generation
- [ ] Show appointment booking
- [ ] Explain notification system

**Quote Generator:**
- [ ] How to create a quote
- [ ] How to send to customer
- [ ] How to save/approve
- [ ] How to track quote status

**Client Practice:**
- [ ] Client generates a test quote
- [ ] Client books a test appointment
- [ ] Client checks notifications

---

### Part 4: Maintenance Training (10 min)

**Monthly Maintenance:**
- [ ] Show maintenance script location
- [ ] Explain when to run it
- [ ] Demonstrate running it
- [ ] Review what it does

**Warning Signs:**
- [ ] Computer running slow
- [ ] Internet acting up
- [ ] Desktop getting messy
- [ ] Programs crashing

**What to Do:**
- [ ] Run monthly maintenance
- [ ] Call Botwave if issues persist

**Client Practice:**
- [ ] Client runs maintenance script
- [ ] Client reads maintenance report

---

### Part 5: Support & Next Steps (10 min)

**Support Information:**
- [ ] Review contact methods
- [ ] Explain response times
- [ ] Show how to request help
- [ ] Review monthly fee

**Next Steps:**
- [ ] Schedule follow-up call (1 week)
- [ ] Schedule monthly checkup
- [ ] Leave printed materials
- [ ] Collect payment

**Client Practice:**
- [ ] Client saves Botwave contact info
- [ ] Client knows how to reach support
